# 교육 프로그램 포스터 & 썸네일 생성기 🎨
> **Education Program Poster & Thumbnail Generator**

교육 프로그램, 세미나, 워크숍 등을 위한 전문가 수준의 포스터(9:16)와 썸네일(16:9) 이미지를 웹 브라우저에서 즉석으로 디자인하고 생성할 수 있는 도구입니다.

## 주요 기능 ✨

- **실시간 미리보기**: 입력한 정보가 포스터와 썸네일 디자인에 즉시 반영됩니다.
- **다양한 레이아웃 지원**: 
  - **포스터**: 스마트폰 최적화 해상도 (1080 × 1920 px, 9:16)
  - **썸네일**: 웹 및 SNS 공유용 해상도 (960 × 540 px, 16:9)
- **세부 커스터마이징**:
  - **포인트 컬러**: 프리셋 컬러 선택 또는 사용자 정의 색상(Hex) 지정 가능
  - **프로그램 안내**: 여러 파트와 세부 항목을 동적으로 추가/삭제 가능
  - **교육 유형**: 일반 강의, 워크숍, 세미나, 심포지엄, 컨퍼런스 등 테마 선택
- **고해상도 다운로드**: 원클릭으로 포스터와 썸네일을 PNG 파일로 저장할 수 있습니다.

## 사용 방법 🚀

1. `index.html` 파일을 크롬, 엣지 등 현대적인 웹 브라우저로 엽니다.
2. 왼쪽 사이드바에서 프로그램 명칭, 날짜, 장소 등 **기본 정보**를 입력합니다.
3. **프로그램 안내** 섹션에서 교육의 상세 커리큘럼(Part 및 세부 항목)을 구성합니다.
4. **포인트 컬러**와 **교육 유형**을 선택하여 브랜드 아이덴티티에 맞는 디자인을 적용합니다.
5. 디자인이 완성되면 하단의 **PNG 저장 ↓** 버튼을 눌러 이미지를 다운로드합니다.

## 기술 스택 🛠️

- **HTML5 / CSS3**: 반응형 레이아웃 및 현대적인 UI 디자인 (Space Grotesk, DM Mono 폰트 사용)
- **JavaScript (Vanilla JS)**: 외부 라이브러리 없이 순수 자바스크립트로 로직 구현
- **HTML5 Canvas API**: 브라우저 내에서 직접 고해상도 그래픽 렌더링 및 이미지 생성
